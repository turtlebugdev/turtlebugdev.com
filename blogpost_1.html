So... I started a thing...

About a week ago I started #100DaysofCode. At the same time I started working my way through The Odin Project.

The Odin Project?

I've been playing around with code for about a decade now. I code queries in SQL everyday at work. In my free time I've done some game programming in Gamemaker and Gamemaker 2. I've also edited a few python queries and completed a few trainings on python on Youtube. I decided that I wanted to utilize my cacity for code for a side gig by doing web development on the side.

I don't have time to go back to school fulltime, nor could I afford to take 12-18 weeks of work financially. So I knew I needed to do this "self-taught". I looked at a bunch of posts about software dev. I started with freecodeacademy for a couple lessons. But I found it wasn't really sticking in my head. I then found app academy open and started going thru their curriculum. It is good, and I may come back to it, specifically for Python. But in the end I found The Odin Project and it really seemed to click. It's a lot more reading and doing stuff on your own (setting up a linux environment for example), but I'm really digging it and I'm learning a lot.
